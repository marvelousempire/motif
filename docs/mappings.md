# WooCommerce to Motif Field Mappings

## Idea Profile

| WooCommerce Field | Motif Label |
| --- | --- |
| Product | Idea |
| Product Name | Idea Title |
| Description | Full Idea Explanation |
| Short Description | Idea Summary |
| SKU | Idea Reference ID |
| Regular Price | Contribution or Value |
| Sale Price | Temporary Value |
| Stock Quantity | Capacity |
| Stock Status | Idea Availability |
| Categories | Idea Categories |
| Tags | Idea Themes |
| Attributes | Idea Traits |
| Variations | Idea Versions |
| Reviews | Feedback |
| Related Products | Related Ideas |
| Upsells | Suggested Ideas |
| Cross-sells | Complementary Ideas |
| Add to Cart | Add to Collection |
| Cart | Collection |
| Checkout | Submit or Support |

## Mapping Rule

Motif should create a semantic label layer above the existing WooCommerce data model.

## Reason

This keeps WooCommerce stable, update-safe, and compatible with existing tools while allowing the user experience to change.
