# Motif Vision

Motif is built around one simple idea:

**Change the lens. Keep the engine.**

WooCommerce already has a strong engine for structured objects. Motif changes the language and experience so those objects can represent ideas, projects, cases, initiatives, opportunities, or other domain-specific records.

## Guiding Idea

A WooCommerce product does not always have to feel like a product.

With Motif, the same underlying object can be presented as an idea.

## First Use Case

The first Motif profile is the Idea profile.

In this profile:

- Products become Ideas
- Categories become Idea Categories
- Tags become Idea Themes
- Attributes become Idea Traits
- Variations become Idea Versions
- Reviews become Feedback
- Cart becomes Collection

## Long-Term Direction

Motif should become a profile-based system where a site owner can activate a domain lens without rebuilding the underlying WordPress and WooCommerce stack.
