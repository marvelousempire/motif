# Indexing Strategy

Indexing is how Motif notes become searchable and retrievable for AI systems.

## Indexing Goals

- Every file should have a clear title.
- Every note should focus on one concept.
- Important terms should appear naturally in the note.
- Related concepts should be linked with wiki links.
- Tables should be used for field mappings.

## Suggested Metadata

Future notes may use frontmatter like this:

```yaml
type: concept
status: draft
motif: idea
related:
  - Motif
  - Semantic Mapping
```
