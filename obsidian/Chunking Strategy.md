# Chunking Strategy

Chunking is the process of splitting knowledge into useful pieces for AI retrieval.

Motif should be written in small Markdown notes so each note becomes a clean chunk.

## Chunking Rules

- Keep one main concept per note.
- Keep paragraphs short.
- Use headings often.
- Use tables for mappings.
- Use wiki links to connect related notes.
- Avoid mixing too many topics in one file.

## Good Chunk Example

A note named [[Semantic Mapping]] explains only semantic mapping.

A note named [[Idea Profile]] explains only the Idea Profile.
