# Motif Obsidian Vault

This folder is designed to work as an Obsidian-friendly wiki layer for Motif.

The goal is to make Motif easy for humans and AI systems to read, link, chunk, index, and retrieve.

## Core Pattern

Motif documentation should use small, linked Markdown notes.

Each note should focus on one idea.

Each note should link to related notes using Obsidian wiki links.

Example:

```md
[[Motif]] connects to [[Semantic Mapping]] and [[WooCommerce Engine]].
```

## Why This Matters for AI

Small linked notes are easier to:

- Chunk
- Embed
- Search
- Index
- Retrieve
- Recombine
- Explain

## Rule

Every major concept should become its own note.

Every note should connect to at least one other note.

Every Motif object must have a valid intent.

Every Motif object should have at least one keyword combo.
