# RAG Strategy

RAG means retrieval augmented generation.

For Motif, the goal is to make every important idea easy to retrieve by an AI system.

## Motif RAG Rules

1. One concept per note.
2. Use clear headings.
3. Use simple language.
4. Link related concepts with Obsidian wiki links.
5. Keep each note useful on its own.
6. Add metadata when needed.

## Retrieval Goal

When an AI asks about Motif, it should be able to retrieve the exact note that explains the concept.
