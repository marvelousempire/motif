# WooCommerce Engine

WooCommerce Engine means the existing WordPress and WooCommerce structure that Motif does not want to break.

It includes:

- Product records
- Product metadata
- Categories
- Tags
- Attributes
- Variations
- Reviews
- Images
- Status
- Visibility
- APIs

Motif keeps this engine intact and changes the language layer above it.
