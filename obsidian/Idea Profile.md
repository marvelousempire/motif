# Idea Profile

Idea Profile is the first Motif lens.

It translates WooCommerce product language into idea language.

## Core Mappings

| WooCommerce | Idea Profile |
| --- | --- |
| Product | Idea |
| Product Name | Idea Title |
| Description | Idea Explanation |
| Short Description | Idea Summary |
| SKU | Idea Reference ID |
| Price | Value or Contribution |
| Stock | Capacity |
| Category | Idea Category |
| Tag | Idea Theme |
| Attribute | Idea Trait |
| Variation | Idea Version |
| Review | Feedback |
